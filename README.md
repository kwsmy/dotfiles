# dotfiles

> [!NOTE]  
> This is written assuming that you *don't* have sudo privileges.  
> See [zsh](https://github.com/kwsmy/dotfiles/tree/zsh) to install useful tools if sudo is available.

## Installation

Symbolic links for dotfiles are created with the following command.

```bash
bash ./scripts/install.bash bash
```

If you want to remove them, run the following command.

```bash
bash ./scripts/uninstall.bash bash
```
